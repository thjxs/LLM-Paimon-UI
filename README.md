# [LLM-Paimon-UI](https://github.com/yuemengrui/LLM-Paimon-UI.git)

## 欢迎访问 Paimon

#### 使用说明：
鼠标移动到页面左侧时，会弹出菜单栏，菜单栏包括首页、对话、应用、知识库等
* 点击应用菜单，在应用页面中创建您的应用
* 在对话菜单中选择应用进行对话
* 在知识库菜单中创建您的知识库, 点击知识库卡片进入详情
